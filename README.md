# restorephotos
根据照片的拍照时间和地理位置信息对图片进行分类。程序原理为读取照片exif信息，然后通过经纬度比较查看距离最近的城市然后放入指定文件夹。<br/>
<br/>
普通用户直接下载可运行程序，解压后运行bat文件即可。<br/>
<br/>
步骤<br/>
1.设置照片文件夹目录<br/>
2.设置要复制照片到哪个目录。<br/>
3.执行<br/>
<br/>
执行结果为：<br/>
将图片按照：<br/>
2017年1月/广州市/xxxx（原始文件名）.jpg<br/>
2017年2月/杭州市/xxxx（原始文件名）.jpg<br/>
2017年3月/深圳市/xxxx（原始文件名）.jpg<br/>
2017年3月/xxxx（原始文件名）.jpg<br/>
